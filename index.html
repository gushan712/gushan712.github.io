🚀 项目经历

📁 宽禁带半导体 GaN 功率器件双脉冲测试研究

项目简介：参与大创/PRP研究项目，针对 EPC2366 等氮化镓（GaN）高频功率器件进行建模与分析。

  • 利用 LTspice 软件搭建双脉冲测试（Double Pulse Test）电路模型。
  • 分析开通/关断瞬态过程中的寄生参数对电压尖峰和开关损耗的影响。
  • 深入探究宽禁带器件在高速开关状态下的暂态稳定性。
📁 基于机器学习的河南某风电场功率预测系统

项目简介：针对复杂气象条件下的风力发电输出功率进行建模预测，提高电网接纳新能源的能力。

  • 使用 Python 对风场历史数据进行清洗，利用 K-means 进行异常数据聚类剔除。
  • 构建并训练 CNN-XGBoost 混合预测模型，实现短期与超短期风电功率预测。
  • 优化预测精度，为功率波动剧烈情况下的电网调度与暂态稳定分析提供数据支撑。
📁 电力系统暂态分析与故障数值模拟

项目简介：基于理论课程与实验扩展，对同步发电机突发短路及系统暂态稳定进行仿真验证。

  • 使用 PLECS/LTspice 建立同步机及多机系统的数学与拓扑模型。
  • 模拟对称与不对称短路故障，应用对称分量法解算各相暂态电流。
  • 运用 Excel/Origin 提取仿真曲线数据，计算关键暂态参数与标准差，撰写完整学术报告。
